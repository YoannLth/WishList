# WishList
iOS 10 app as part of a Udemy tutorial.
Wishlist iOS app written in Swift and using CoreData.

## Learning goals
* Work with CoreData
* Work with UIKit components (UIPickerView, UITableView, UIImagePickerController...)
* Data sorting in table view
